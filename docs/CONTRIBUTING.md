# Contributing to JustStorage

Thanks for your interest! This document outlines how to contribute to the project.

1. Read `DESIGN.md` and `IMPLEMENTATION.md` to understand the architecture.
2. Open an issue describing your feature or bug (or check for existing issues first).
3. Fork, branch, and implement your change. Keep changes small and focused.
4. Add tests for new behavior, and run `cargo test` under `rust` to verify.
5. Ensure `cargo fmt` and `cargo clippy` pass before requesting a review.
6. Create a pull request and reference the issue. Use clear, descriptive PR titles.

Code of Conduct
----------------

Please follow the project's `CODE_OF_CONDUCT.md` for community behavior expectations.
