# Security Policy

If you discover a security vulnerability in this project, please report it
privately to [security@glowing-pixels.com](mailto:security@glowing-pixels.com) or open a GitHub Security
Advisory via the repository's Security tab.

Please provide:

- Affected package and version(s)
- A clear description of the issue and impact
- Minimal repro steps or a PoC
- Any relevant logs or stack traces

We will acknowledge receipt within 72 hours and work with you on coordinated
disclosure.

For non-security questions, please use the project's Discussions forum or 
open a regular issue.
